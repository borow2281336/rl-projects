# RL Blackjack Homework

Учебный проект по обучению агента играть в Blackjack с помощью табличного Q-learning.

## Описание

В проекте рассматривается игра Blackjack в среде `Blackjack-v1` из библиотеки Gymnasium.  
Агент обучается принимать решения **взять карту / остановиться**, максимизируя ожидаемый суммарный выигрыш.

Реализованы:

- табличный Q-learning;
- ε-жадная стратегия с фиксированным ε;
- ε-жадная стратегия с убывающим ε (decaying epsilon);
- сравнение качества полученных политик;
- визуализация процесса обучения (график скользящего среднего вознаграждения);
- текстовая демонстрация одного эпизода с выводом карт игрока и дилера.

Основной код и отчёт находятся в ноутбуке:

- `blackjack_rl.ipynb` (название можно поменять на твоё реальное)
